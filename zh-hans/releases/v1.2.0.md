## 版本说明

此版本中最显著的变化包括：

* 性能优化
* 提升与 Go 1.9 的兼容性
* 支持创建堆积条形图
* 支持 24 小时格式时间
* API 更改, 使用动态工作表名称代替 "sheet" + index 格式的工作表名称
* 代码库地址变更为 `github.com/360EntSecGroup-Skylar/excelize`
* 新增函数 `SetSheetViewOptions` 和 `GetSheetViewOptions` 支持设置和获取工作表的视图属性
